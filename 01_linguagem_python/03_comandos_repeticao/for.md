# Comando for 

~~~python
for i in range(10): # de 0 a 9
    print(i)
~~~

~~~python
for i in range(1, 11):
    print(i)
~~~

~~~python
for x in range(6):
    print(x)
else:
    print("Saí do for") 
~~~

## Iterando por uma lista

~~~python
lista_convidados = ["João", "Maria", "José"]

for convidado in lista_convidados:
    print(convidado)

# outra forma
for i in range(len(lista_convidados)):
    print(lista_convidados[i])
~~~

## Operador Condicional (Operador Ternário)

~~~python
min = a if a < b else b
~~~

~~~python
print("a é maior") if a > b else print("b é maior")
~~~

~~~python
print("a é maior") if a > b else print("a igual a b") if a == b else print("b é maior")
~~~

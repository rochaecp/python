# Python - Tipos de Dados

- Numéricos
    - ```int```
    - ```float```
    - ```complex``` (ex.: 1j) 
- String
    - ```str```
- Booleano
  - ```bool```
- Binário
    - ```bytes```
    - ```bytearray```
    - ```memoryview```
- Sequências
    - ```list```
    - ```tuple```
    - ```range```
- Dicionários 	
    - ```dict```
- Conjuntos
    - ```set```
    - ```frozenset```

## Obtendo o tipo de uma variavel

~~~python
myType = type(minhaVariavel)
~~~

## Tipos numéricos

~~~python
x = 1    # int
y = 2.8  # float
z = 1j   # complex
~~~
